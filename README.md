## Langkah yang dilakukan
1. Membuat repository baru di GitHub bernama tugas-github.
2. Membuat file README.md secara otomatis melalui GitHub.
3. Melakukan initial commit.
4. Menghubungkan repository lokal dengan repository GitHub.
5. Melakukan push commit awal.
